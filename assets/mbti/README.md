# MBTIキャラクター画像の置き場所

このフォルダに以下の16ファイルを置くと、トップページの
「新しい就活イベント」カードの背景として順番に表示されます。
1体あたり約1.25秒、16体で約20秒かけて一周します。

必要なファイル（すべてPNG・背景透過を推奨）:

  INTJ.png  INTP.png  ENTJ.png  ENTP.png
  INFJ.png  INFP.png  ENFJ.png  ENFP.png
  ISTJ.png  ISFJ.png  ESTJ.png  ESFJ.png
  ISTP.png  ISFP.png  ESTP.png  ESFP.png

推奨サイズ: 縦600px前後（横は自由）。背景透過のPNGだと最もきれいに出ます。

1枚でも置けば画像モードに切り替わり、置いていない間は
タイプ名のテキスト表示のままになります（表示が壊れることはありません）。

注意: 他者が作成したイラストを許諾なく使用しないでください。
ご自身で生成・作成した画像か、商用利用が許諾された素材を使ってください。
